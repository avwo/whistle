# 快捷键

## Network
1. `Ctrl[Command] + I`: 导入抓包数据
2. `Ctrl[Command] + E`: 导出抓包数据
3. `Ctrl[Command] + S`: 保存抓包数据
4. `Ctrl[Command] + O`: 关闭或打开抓包
5. `Ctrl[Command] + L`: 切换 Network 面板的左右或上下布局
6. `Ctrl[Command] + .`: 打开 Settings
7. `Ctrl[Command] + D`: 删除选中的抓包数据
8. `Ctrl[Command] + B`: 切换抓包数据的树形或列表形式
9. `Ctrl[Command] + Enter`: 重新请求选中的抓包数据
10. `Ctrl[Command] + Shift + Enter`: 设置重新请求选中抓包数据的次数
11. `Ctrl[Command] + A`: 强制中断选中的请求
12.  `Ctrl[Command] + X`: 清空抓包数据
13. `/ `: 聚焦到底部搜索框


## Frames
1. `Ctrl[Command] + Enter`: 重新请求 Frames 面板选中的请求数据
2. `Ctrl[Command] + X`: 清空 Frames 面板的数据

## Rules
1. `Ctrl[Command] + I`: 导入 Rules
2. `Ctrl[Command] + E`: 导出 Rules
3. `Ctrl[Command] + S`: 保存 Rules
4. `Ctrl[Command] + O`: 关闭或打开 Rules
5. `Ctrl[Command] + L`: 切换显示编辑器的行数
6. `Ctrl[Command] + .`: 打开 Settings
7. `Ctrl[Command] + D`: 删除选中的 rules
8. `/ `: 聚焦到底部搜索框

## Values
1. `Ctrl[Command] + I`: 导入 Values
2. `Ctrl[Command] + E`: 导出 Values
3. `Ctrl[Command] + S`: 保存 Values
4. `Ctrl[Command] + L`: 切换显示编辑器的行数
5. `Ctrl[Command] + .`: 打开 Settings
6. `Ctrl[Command] + D`: 删除选中的 values
7. `/ `: 聚焦到底部搜索框


## Plugins
1. `Ctrl[Command] + I`: 打开安装插件对话框
2. `Ctrl[Command] + O`: 关闭或打开所有 Plugins

## Global
1. `Ctrl[Command] + <--`: 反向切换 Network、Rules、Values、Plugins
2. `Ctrl[Command] + -->`: 切换 Network、Rules、Values、Plugins

## Client
1. `Ctrl[Command] + R`：打开或关闭系统代理
2. `Ctrl[Command] + Shift + R`：重启客户端
3. `Ctrl[Command] + Q`：退出客户端
4. `Ctrl[Command] + ,`：打开 Proxy Settings
